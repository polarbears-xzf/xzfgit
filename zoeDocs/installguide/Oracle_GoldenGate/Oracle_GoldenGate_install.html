Oracle_GoldenGate的安装

  • 安装Oracle版本
    Oracle版本12.2.0.1版本
  • 安装Oracle_GoldenGate版本
    ogg版本为12.3.0.15（123015_ggs_Windows_x64_shiphome）
    安装过程中，需要指定Oracle数据版本是12c还是11g
  • 需要其他插件
    需要vcredist_x64.exe插件
    否则运行ggsci.exe会提示缺少动态库
